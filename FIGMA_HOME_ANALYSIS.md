# Figma Home Page Analysis

Источник: `https://www.figma.com/design/dC7JG8bynf2eE29eVuMhuV/...node-id=2056-401`

Фрейм: `Главная / Стадия 3`

## Общая картина

Главная страница для Кубэра построена как длинный лендинг о производстве столярных изделий из массива. Визуальная основа: темный фон, теплые древесные оттенки, акцентный золотой цвет, крупные фотографии изделий и мастерской.

## Секции страницы

1. Header
   - Логотип.
   - Меню: Услуги, О компании, Отзывы, Контакты.
   - Телефон.
   - Email.
   - Кнопка "Заказать звонок".

2. Hero
   - Надзаголовок: "Столярные изделия".
   - Заголовок: "Производство эксклюзивных столярных изделий из массива".
   - Карточки преимуществ: "15 лет мастерства", "Индивидуальный подход", "От дизайна до установки".

3. Услуги
   - Карточки категорий:
     - Наличники.
     - Лестницы.
     - Порталы и арки.
     - Обсады и подоконники.
     - Декор.
   - Отдельная CTA-карточка с предложением рассчитать стоимость или оставить заявку.

4. Запрос цены
   - Блок "Запросите цену на любой объект".
   - Карточки типов объектов или направлений.
   - Кнопка заявки.

5. Реализованные проекты
   - Заголовок "Уже реализованные проекты".
   - Список проектов с фото.
   - Кнопка просмотра всех проектов.

6. О мастерской
   - Блок про собственное производство и столярную мастерскую.
   - Фото мастерской.
   - Список фактов/преимуществ.

7. Отзывы
   - Заголовок "Мнение наших клиентов".
   - Карточки отзывов.
   - Навигация/слайдер.

8. Особенности работы
   - Карточки преимуществ работы с компанией.

9. FAQ
   - Блок "Ответы на часто задаваемые вопросы".
   - Список раскрывающихся вопросов.

10. Форма заявки
    - Заголовок "Остались вопросы?"
    - Контактное лицо.
    - Соцсети.
    - Поля: имя, телефон, комментарий.
    - Согласие с политикой конфиденциальности.

11. Как добраться
    - Адрес.
    - Время работы.
    - Кнопка "Построить маршрут".
    - Карта.

12. Footer
    - Юридическая информация.
    - Телефон, почта, соцсети.
    - Список услуг.
    - Документы.
    - Копирайт и ссылка на разработчика.

## Что должно редактироваться в WordPress

### Глобальные настройки

- Логотип.
- Телефон.
- Email.
- Адрес.
- Время работы.
- Соцсети.
- Юридическая информация.
- Ссылки на документы.
- Текст кнопок заявок.

### Главная страница

- Hero: надзаголовок, заголовок, фон/декоративные изображения.
- Преимущества в hero.
- Список услуг с заголовками, описаниями, изображениями и ссылками.
- CTA-карточки.
- Список объектов для запроса цены.
- Реализованные проекты.
- Блок о мастерской.
- Отзывы.
- Особенности работы.
- FAQ.
- Форма обратной связи.
- Карта и ссылка маршрута.

## Рекомендуемая WordPress-структура

### Типы записей

- `service` - услуги.
- `project` - реализованные проекты.
- `review` - отзывы.
- `lead` - заявки, если решим хранить формы без стороннего плагина.

### Поля ACF

- Группа `Theme Settings` для контактов, соцсетей, документов, юридической информации.
- Группа `Home Page` для секций главной.
- Группа `Service Details` для страниц услуг.
- Группа `Project Details` для проектов.
- Группа `Review Details` для отзывов.

### Шаблоны темы

- `front-page.php` - главная.
- `header.php` - шапка.
- `footer.php` - подвал.
- `template-parts/sections/*` - секции главной.
- `single-service.php` - страница услуги.
- `archive-service.php` - список услуг.
- `single-project.php` - проект.
- `archive-project.php` - список проектов.

## Технические заметки по верстке

- Базовая ширина макета: 1920 px.
- Основные отступы desktop: 40 px от краев.
- Основной фон: `#170f0b`.
- Цвет карточек: `#211b17`.
- Акцент: `#b57900`.
- Основные шрифты из макета: Arial, Russo One, Geist.
- Нужно отдельно адаптировать mobile/tablet, потому что текущий фрейм передан как desktop-длинная страница.

## Ближайший рабочий шаг

Создать базовую структуру темы WordPress и статически сверстать главную страницу по секциям. После этого подключить редактируемые поля и типы записей.
